# Schema

## Schema

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| name | `string` | Yes | A string value |
| age | `number` | Yes | A numeric value |
| active | `boolean` | Yes | A boolean value (true/false) |

## Example

```json
{
  "name": "example",
  "age": 123,
  "active": true
}
```
